# Automation Framework (UI + API)

## 📌 Description

Automation framework covering:

* UI testing (Selenium)
* API testing (RestSharp)

## 🧪 Projects

### 🔹 SeleniumAPI

* API tests for '\/client\'
* NUnit + RestSharp
* JSON Schema validation

### 🔹 SeleniumTests

* UI automation using Selenium WebDriver
* Page Object Model (POM)

## 🛠 Tech Stack

* C#
* .NET
* NUnit
* Selenium WebDriver
* RestSharp
* Newtonsoft.Json

## 🚀 Features

* Config-driven environment
* Positive & Negative test cases
* Reusable API client
* Clean architecture

## ▶️ Run tests

```bash
dotnet test
```
